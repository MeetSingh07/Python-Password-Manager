# Python Password Manager

A beginner-friendly command-line password manager built using Python.

This project can generate strong passwords, save them in a file, store custom passwords, and retrieve saved passwords using exact-match searching.

---

## Features

- Generate random strong passwords
- Custom password length
- Save passwords to a file
- Store custom passwords manually
- Retrieve saved passwords
- Exact-match password searching
- Duplicate password prevention
- Input validation using exception handling
- Menu-driven interface

---

## Technologies Used

- Python
- File Handling
- Exception Handling
- Functions
- Loops & Conditions
- Random Module

---

## How to Run

1. Clone the repository

```bash
git clone YOUR_REPOSITORY_LINK
```

2. Open project folder

3. Run the Python file

```bash
python main.py
```
